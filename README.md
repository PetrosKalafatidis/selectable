# selectable
An angularjs selectable directive, creates a selectable item you can optionally use an active-class attribute.

```html
<ul class="tab-nav">
    <li pn-selectable active-class="active" class="active">
        <a href="#">
            link1
        </a>
    </li>
    <li pn-selectable active-class="active">
        <a href="#">
            link2
        </a>
    </li>
</ul>
```
